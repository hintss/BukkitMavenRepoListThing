What is this you ask?

This is a list of maven repos + dependency names for various libs/things used in bukkit plugin coding. More to be used as an index for me than for public use, but it'll probably be useful to someone. Just ctrl+f for the lib you need, I guess.

Bukkit

http://bukkit.org/ http://jd.bukkit.org/dev/apidocs/

<repository>
  <id>bukkit-repo</id>
  <url>http://repo.bukkit.org/content/groups/public/</url>
</repository>

<dependency>
  <groupId>org.bukkit</groupId>
  <artifactId>bukkit</artifactId>
  <scope>provided</scope>
  <version>RELEASE</version>
</dependency>
